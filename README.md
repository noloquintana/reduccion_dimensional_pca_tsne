# ESPAÑOL

# 📊 Reducción de Dimensionalidad con PCA y t-SNE

---

## 📌 Descripción

Este proyecto tiene como objetivo aplicar técnicas de reducción de dimensionalidad, específicamente **PCA (Análisis de Componentes Principales)** y **t-SNE**, sobre un dataset de alta dimensión para facilitar su visualización, análisis e interpretación.

Se trabaja con el dataset **Human Activity Recognition (HAR)**, el cual contiene datos obtenidos a partir de sensores de smartphones, con más de 500 variables.

---

## 🎯 Objetivos

- Reducir la dimensionalidad del dataset
- Visualizar los datos en 2 dimensiones
- Comparar PCA y t-SNE
- Identificar patrones o agrupaciones
- Evaluar qué técnica es más adecuada para el análisis

---

## 🧠 Tecnologías utilizadas

- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## 📂 Dataset

**Nombre:** Human Activity Recognition (HAR)  
**Fuente:** https://archive.ics.uci.edu/dataset/240/human+activity+recognition+using+smartphones  

El dataset contiene **561 variables** y miles de observaciones, lo que lo hace ideal para aplicar técnicas de reducción de dimensionalidad.

---

## ⚙️ Metodología

1. Carga de datos  
2. Exploración inicial  
3. Escalamiento de variables (StandardScaler)  
4. Aplicación de PCA  
5. Aplicación de t-SNE  
6. Visualización de resultados  
7. Análisis comparativo  
8. Conclusiones  

---

## 📊 Resultados

- **PCA** permite visualizar la estructura global de los datos, pero no separa claramente los grupos.  
- **t-SNE** muestra agrupaciones más definidas, facilitando la identificación de patrones.  

---

## 🧩 Conclusiones

t-SNE es más adecuado para la visualización de segmentos o grupos en los datos, especialmente en contextos como marketing o comportamiento de usuarios.  
PCA es útil como paso previo debido a su eficiencia y simplicidad.

# ENGLISH

# 📊 Dimensionality Reduction with PCA and t-SNE

---

## 📌 Description

This project aims to apply dimensionality reduction techniques, specifically **PCA (Principal Component Analysis)** and **t-SNE**, on a high-dimensional dataset to facilitate visualization, analysis, and interpretation.

The project uses the **Human Activity Recognition (HAR)** dataset, which contains data collected from smartphone sensors and includes more than 500 features.

---

## 🎯 Objectives

- Reduce the dimensionality of the dataset  
- Visualize the data in 2 dimensions  
- Compare PCA and t-SNE  
- Identify patterns or clusters  
- Evaluate which technique is more suitable for analysis  

---

## 🧠 Technologies Used

- Python 🐍  
- NumPy  
- Pandas  
- Matplotlib  
- Scikit-learn  

---

## 📂 Dataset

**Name:** Human Activity Recognition (HAR)  
**Source:** https://archive.ics.uci.edu/dataset/240/human+activity+recognition+using+smartphones  

The dataset contains **561 features** and thousands of observations, making it ideal for applying dimensionality reduction techniques.

---

## ⚙️ Methodology

1. Data loading  
2. Initial exploration  
3. Feature scaling (StandardScaler)  
4. PCA application  
5. t-SNE application  
6. Visualization of results  
7. Comparative analysis  
8. Conclusions  

---

## 📊 Results

- **PCA** helps visualize the global structure of the data but does not clearly separate clusters.  
- **t-SNE** reveals more defined clusters, making it easier to identify patterns.  

---

## 🧩 Conclusions

t-SNE is more suitable for visualizing groups or segments in the data, especially in contexts such as marketing or user behavior analysis.  
PCA is useful as a preprocessing step due to its efficiency and simplicity.
